# RFC 0001 — Hyphence (moved to hyphence)

This RFC — the normative specification for the hyphence (`---`-fenced
metadata + body) document format — was re-homed to the **hyphence**
repository, which now owns the format specification and its
implementations (Go, with Rust to follow). It keeps its number, RFC
`0001`, in the new repo. It moved out of madder as part of the hyphence
extraction ([madder#253]); madder continues to use the format but no
longer owns its specification.

The canonical, maintained version lives at:

<https://github.com/amarbel-llc/hyphence/blob/master/docs/rfcs/0001-hyphence.md>

This stub is a redirect for old links. See the madder git history for the
prior in-tree content.
